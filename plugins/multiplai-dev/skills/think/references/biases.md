# Bias Checklist

Scan actively for these. When you identify one: name it, explain briefly why you suspect it, ask a question that tests whether it's operating.

## Inertia Biases (resistance to change)

| Bias | Signal |
|------|--------|
| **Status quo** | Preference for current state inflated by familiarity |
| **Endowment effect** | Overvaluing what we already have |
| **Sunk cost** | Past investments distorting forward-looking decisions |
| **Plan continuation** | Sticking with original plan despite changed circumstances |
| **Loss aversion** | Potential loss weighted more than equivalent gain |

## Framing Biases (how the problem is presented)

| Bias | Signal |
|------|--------|
| **Anchoring** | Early number/option dominating the frame |
| **Framing effect** | Would reframing same facts change the conclusion? |
| **Availability** | Recent/vivid examples distorting probability estimates |
| **Base rate neglect** | Ignoring general statistics for specific case details |

## Confidence Biases (over/under-estimation)

| Bias | Signal |
|------|--------|
| **Overconfidence** | Certainty exceeding what evidence supports |
| **Optimism bias** | Underestimating probability of bad outcomes |
| **Pessimism bias** | Overweighting negative possibilities |
| **Planning fallacy** | Underestimating time/cost/difficulty |
| **Illusion of control** | Believing we can influence things we can't |

## Evaluation Biases (judging outcomes)

| Bias | Signal |
|------|--------|
| **Confirmation** | Seeking/favoring evidence supporting existing beliefs |
| **Survivorship** | Only seeing successes, not failures |
| **Outcome bias** | Judging decision quality by result, not process |
| **Hindsight** | Believing past events were more predictable |

## Social/Attribution Biases

| Bias | Signal |
|------|--------|
| **Fundamental attribution error** | Overweighting personality vs. situation |
| **Self-serving** | Credit for success, blame circumstances for failure |
| **Groupthink** | Conforming to consensus without critical evaluation |

## Action Biases

| Bias | Signal |
|------|--------|
| **Action bias** | Feeling compelled to act when inaction might be better |
| **Omission bias** | Preferring inaction because it feels less responsible |
| **Hyperbolic discounting** | Overweighting immediate payoffs vs. future ones |
